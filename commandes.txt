1) créer un environnement virtuel: python -m venv nom_env

2) activer l'environnement virtuel: source nom_env/Scripts/activate

3) installer Flask: pip install flask

4) configurer le fichier à lancer pour lancer l'app: export FLASK_APP=nom_fichier_python 
pour lancer en mode développement : export FLASK_ENV=development

5) lancer l'app: flask run

6) utiliser des templates html: render_template('code_html')

la fonction url_for permet à partir de ses arguments de générer un path ou une url

{% block content %} {% endblock %}: quand on recupere un code html par héritage, la partie dans un block peut etre personnalisée.
Pour hériter d'un fichier html père: {% extends 'nom_pere.html' %}


   <p ><select name= "choices">
    {% for value in queries %}
    <option>query{{value}}</option>
    {% endfor %}
</select></p>
